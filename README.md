# ETRI AX Strategy Team

ETRI AX전략팀 소개용 웹 콘텐츠와, 실험용 Prompt Upgrader 프로젝트를 함께 담은 작업 공간입니다.

## What is inside

- `webpage/`
  - ETRI AX전략팀 소개 웹사이트
  - 정적 HTML/CSS로 바로 열어볼 수 있는 원페이지 랜딩
- `prompt generator/`
  - Next.js 기반 Prompt Upgrader 실험 프로젝트

## How to view the website

1. `webpage/index.html`을 브라우저로 엽니다.
2. 같은 폴더의 `styles.css`가 자동으로 적용됩니다.

## Local notes

- GitHub에는 `main` 브랜치로 올라가 있습니다.
- 소개서 원본 문서는 `webpage/*.docx` 규칙으로 제외했습니다.
